-- Student Management System (Python) --

Command-line student management system built with Python.

#Features

- Add new students
- Assign subjects and grades
- Calculate:
  - General average
  - Average by subject
- Identify:
  - Approved students
  - Failed students
  - Best student
- Store data in a file

---

#Technologies

- Python 3
- File handling
- Dictionaries & lists
- CLI interaction

- Add student
- Add grades
